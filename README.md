# Particles-W-Hand

## Description

Moving particles with your hand (index finger actually)

## Dependencies

- Pygame
- Opencv
- Mediapipe

## How To Run

1. Clone this repo
2. Navigate to the repo directory
3. Make sure you have all the dependencies installed (I recommend to use virtual environment)
```bash
pip install requirements.txt
```
4. Run the simulation
```bash
python main.py
```
